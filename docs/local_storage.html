Mixin: local_storage

local_storage

This mixin implements a persistence storage feature which stores items on web browser's localStorage. On environments which do not support localStorage, such as node.js, this simulates a localStorage but it's volatile.
Source:
  • storage/local_storage.js, line 3
See:
  • http://dev.w3.org/html5/webstorage/#the-localstorage-attribute

Example

var opts = {
  storage: 'local_storage',
  namespace: 'myspace'
};

var collection = new KagoDB(opts);

collection.read('foo', function(err, item){
  console.log(item);
});
